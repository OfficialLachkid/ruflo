## Premium checklist

Before reporting the site done, verify all of these:

- [ ] Every section renders on `paper` — no dark inverse zones anywhere.
- [ ] All headings use the same serif font. No sans display used.
- [ ] Mono only appears in eyebrows, captions, split-flap values, and legal.
- [ ] Every image has a visible mono `Fig. XX — caption · Photograph` underneath.
- [ ] Nav is a thin editorial bar, not a pill.
- [ ] The split-flap widget is present in the hero byline AND (smaller) in the footer, and nowhere else.
- [ ] Manifesto has one accent-colored underline that draws on scroll-in.
- [ ] Feature Splits alternate sides. Image aspect is portrait (4:5).
- [ ] Services are a numbered list (not a grid), with hairline dividers.
- [ ] Testimonial is a single big pull-quote — not a grid of cards.
- [ ] CTA banner has ONE CTA. Not two. Not three.
- [ ] Footer closing line says "Set in [Font] · Published in [City] · MMXXVI · [Company]".
- [ ] Console: zero errors.
- [ ] Mobile 375: all sections stack; nav becomes hamburger; feature splits become vertical (image above text); index list rows still readable; split-flap is legible.

If any box is unticked, fix it before reporting done.
