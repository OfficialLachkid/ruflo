# Industry Themes

Use this to reskin the v3 signal-map and supporting accents.

| Industry | Signal-map motif | Accent direction |
|---|---|---|
| Electrical | circuit route, spark nodes | electric yellow + cyan |
| Construction | site-plan route, structure markers | safety orange + graphite |
| Finance | trust checkpoints, clean graph route | navy + gold |
| Legal | case-route markers, precise linework | ink blue + brass |
| Hospitality | guest journey path, destination pins | deep teal + sand |
| Real estate | property grid route, location markers | slate + copper |
| Health / wellness | calm pulse path, soft nodes | eucalyptus + warm stone |
| SaaS / tech | network route, data nodes | indigo + neon mint |
| Automotive | performance route, mechanical markers | charcoal + amber |
| Creative studio | gallery route, frame markers | black + silver or rose bronze |

Label the proof ribbon and trust chips to match the industry language.

Examples:

- legal: clarity, diligence, response time
- construction: delivery, site coordination, compliance
- hospitality: response, comfort, booking flow
- SaaS: velocity, reliability, onboarding
