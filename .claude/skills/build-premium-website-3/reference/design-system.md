# Design System

## Core identity

v3 should feel like a premium showroom:

- cinematic
- measured
- spatial
- modern
- not magazine
- not generic SaaS

## Typography

- display: `Space Grotesk`
- body: `Manrope`
- utility and metadata: `IBM Plex Mono`

## Palette

Use:

- one primary color
- one accent color
- one deep surface color
- one warm or cool neutral background

Avoid rainbow palettes or too many highlight colors.

## Surfaces

Default structure:

- page background: soft neutral
- hero / CTA surfaces: deep tone
- content panels: layered translucent surfaces or muted blocks
- borders: hairlines or low-contrast separators

## Motion feel

Motion should feel:

- slow enough to read
- purposeful
- premium
- physically believable

Motion should not feel:

- bouncy
- playful
- noisy
- overloaded

## Tone variants

### architectural-minimal

- lower contrast
- stone, graphite, sand, muted brass

### kinetic-modern

- darker base
- sharper accent
- more pronounced signal-map pulse

### warm-premium

- warmer neutrals
- softer shadows
- bronze or amber accents

### technical-confidence

- stronger grids
- cooler tones
- clearer structural lines
