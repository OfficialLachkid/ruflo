# Intake Questions

Use these as the default intake flow for v3.

## Round 1

- What is the company name?
- What is the shortest strong promise line for the business?
- What does the business do in one sentence?
- Which tone fits best:
  - architectural-minimal
  - kinetic-modern
  - warm-premium
  - technical-confidence

## Round 2

- What primary color should the brand lean on?
- What accent color should support it?
- List 5-7 services.
- For each service, give a short outcome line, not just a label.

## Round 3

- What are the main trust signals?
  - years
  - certifications
  - notable clients
  - region served
- What is the main CTA?
- What contact info should appear?

## Round 4

- Give 3 showcase or proof modules:
  - title
  - short summary
  - image search keywords
- What language should the site use?
- What hero image keywords should be used if real images need to be sourced?

If the user answers sparsely, infer defaults and continue.
